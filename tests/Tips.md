- **why use let/ const**
    - const = hang số = ko thay doi duoc => thay doi => bao loi
    - let = bien = thay doi duoc
> luon luon dung const, chi dung let khi co chủ đích
```typescript
let d = 5;
d = 10;
d = 100;
d = 200;

const PI = 3.14; 
PI = 3.15 // error

let PI2 = 3.14
PI2 = 3.15 // no error
```
